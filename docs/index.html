131 endpoints · 4 layers · Real-time market intelligence

Authentication: All paid endpoints require an X-API-Key header. Create a key via POST /api/keys/create with your wallet address. Briefs are free — no key required on the Telegram channel. 
# Get your API key
curl -X POST https://api.agentcanary.ai/api/keys/create \
  -H "Content-Type: application/json" \
  -d '{"walletAddress": "0xYOUR_WALLET"}'

# Check macro regime
curl -H "X-API-Key: YOUR_KEY" \
  https://api.agentcanary.ai/api/macro/regime

# DeFi P/E ratios (unique to AgentCanary)
curl -H "X-API-Key: YOUR_KEY" \
  https://api.agentcanary.ai/api/defi/pe-ratios
